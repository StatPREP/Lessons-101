StatPREP Little App Activities

These are lessons and other resources for teaching a conventional Stat 101 course, but in a way that puts data at the center.

Use the menu items in the header to access:

  • Little Apps: Interactive statistics displays
  • Instructor notes on topics in Stat 101. These generally encompass more than one lesson.
  • Student-facing activities that build on the Little Apps

At the bottom of each activity file is a link to an editable word-processing file containing the activity. Instructors who want to customize an activity can download this file via the link.

This site is best viewed in Chrome, Firefox, or Safari. Sorry Internet Explorer fans!
